# G3 Architects

G3 Architects is a modern and responsive landing page designed for an architecture firm.  
The website showcases company features, team members, achievements, and sponsors with a clean and professional layout.

## Project Overview
This project is built using semantic HTML and custom CSS, focusing on responsive design and user-friendly UI.  
It is suitable for startup architecture firms or portfolio demonstrations.

## Features
- Responsive layout for desktop, tablet, and mobile devices
- Modern and clean UI design
- Feature highlights and experience showcase
- Team section with image grid
- Facts and statistics section
- Sponsor logos with grayscale hover effect

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- Google Fonts (Work Sans)
